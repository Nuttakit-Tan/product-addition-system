# Product Addition System

## Overview

Product Addition System is a workflow automation solution built using Microsoft Power Platform to manage product creation and modification requests across multiple departments.

This system streamlines product approval, accounting mapping, IT configuration, and supervisor verification processes into a centralized digital workflow.

---

## Technologies Used

* Power Apps (Canvas App)
* Power Automate
* SharePoint Online
* Excel Online
* Microsoft Planner

---

## Key Features

* Add / Edit product requests
* Multi-level approval workflow
* Excel import validation
* Accounting product mapping
* IT task automation
* Supervisor verification
* Automated PDF document generation

---

## Workflow Process

1. User submits product request via Power Apps
2. Data stored in SharePoint List
3. Approval workflow triggered
4. Accounting performs product mapping
5. IT creates product in company system
6. Supervisor reviews completion
7. System archives final document
